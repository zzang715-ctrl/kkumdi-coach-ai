# 꿈디코치 AI 강사비서 출시 요약

## 현재 판단

현재 버전은 v1.0 기본 버전입니다.

화면과 기본 업무 흐름은 갖춰졌습니다. 실제 공개 전에는 OpenAI 결제, Supabase 서버 저장, Vercel 배포 테스트가 마지막으로 필요합니다.

## 준비된 기능

- 로그인 없이 브라우저 저장소로 프로젝트 테스트
- 새 프로젝트 만들기와 기본 정보 수정
- 저장된 프로젝트 검색, 정렬, 복제, 삭제
- 제안서부터 다운로드까지 8단계 업무 흐름
- 각 단계 샘플 초안 만들기, 복사, 저장
- 저장 중, 저장됨, 저장 필요 상태 표시
- 프로젝트 결과물 미리보기와 전체 복사
- Markdown, Word, PDF 다운로드 흐름
- OpenAI 연결 점검과 quota 안내
- Supabase 테이블, RLS, 연결 점검 안내
- Vercel 배포 준비 안내
- 체크 가능한 QA와 QA 결과 내보내기

## 공개 전 남은 일

1. OpenAI 결제와 실제 생성 확인
2. Supabase 실제 프로젝트 연결
3. Vercel 실제 배포
4. 사용자 테스트

## 추천 진행 순서

1. QA 체크리스트를 한 번 끝까지 눌러 봅니다.
2. Supabase 프로젝트를 만들고 SQL을 실행합니다.
3. `.env.local`에 Supabase 값을 넣고 로그인/서버 저장을 확인합니다.
4. OpenAI 결제와 API 사용량을 확인합니다.
5. AI 생성 결과를 실제 강의 예시로 점검합니다.
6. Vercel에 배포하고 배포 주소에서 다시 QA를 진행합니다.

## 출시 요약 저장

앱의 `http://localhost:3000/release-notes` 화면에서 `출시 요약 내보내기` 버튼을 누르면 Markdown 파일로 저장할 수 있습니다.
